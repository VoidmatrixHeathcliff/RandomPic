### 开箱即用的 `随机图片` 服务器

本项目基于 [EtherEngine](https://github.com/VoidmatrixHeathcliff/EtherEngine) 搭建而成，启动后访问可以返回随机图片

你可以：

+ 在 image 文件夹中添加你希望返回的图片

+ 在 scripts 中查看脚本并自定义服务器逻辑

+ ~~将你的 █ 图服务器的链接发送给群友~~
